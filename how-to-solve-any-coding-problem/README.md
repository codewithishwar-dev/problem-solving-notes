# 🚀 How to Solve Any Coding Problem

Most developers get stuck not because the problem is hard...

But because they don’t have a clear approach.

This guide shares a simple and effective **7-step method** to solve any coding problem.

---

## 🧠 The 7-Step Approach

### 1. Understand the Problem
- Read the problem carefully
- Don’t rush into coding
- Clarify constraints and requirements

### 2. Identify Input & Output
- What are you given?
- What do you need to return?

### 3. Start with Brute Force
- Think of the simplest possible solution
- Focus on correctness first

### 4. Optimize Step-by-Step
- Improve your solution gradually
- Use patterns:
  - HashMap
  - Two Pointers
  - Stack
  - Sliding Window

### 5. Write Clean Code
- Keep it simple and readable

### 6. Dry Run Your Solution
- Test with examples

### 7. Debug Logically
- Trace step by step
- Fix without panic
